## Chay rieng frontend

Tu thu muc `frontend`:

```powershell
npm install
npm run dev
```

Frontend mac dinh chay tai `http://localhost:5173` va goi API ve backend `http://localhost:8080` (xem `axiosConfig.ts`).

## Script co san

```powershell
npm run dev
npm run build
npm run preview
npm run lint
```
